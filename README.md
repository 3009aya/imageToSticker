## Getting Started

This project require NodeJS v12.

### Install
Clone this project

```bash
> git clone https://github.com/YogaSakti/imageToSticker.git
> cd imageToSticker
```

Install the dependencies:

```bash
> npm install
```

### Usage
1. run the Whatsapp bot

```bash
> npm start
```
